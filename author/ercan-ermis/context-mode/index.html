// architecture

How It's Built

A multi-language MCP server: Rust executor for sandboxed command isolation, Elixir core for FTS5 indexing and search, exposed as MCP tools.

Rust
Sandboxed Executor
Runs shell commands inside an isolated process group. Enforces timeouts, kills entire process trees on exit, prevents sandbox escapes. Built with Claude reviewing every security boundary.
process group timeout kill nix crate
Elixir / SQLite
FTS5 Index
Command output is chunked and indexed in SQLite FTS5 tables. BM25-ranked queries retrieve only the relevant fragments — the rest stays in the sandbox, never in the context.
FTS5 BM25 rank Exqlite
MCP Protocol
Tool Interface
Exposes batch_execute, search, execute, execute_file, and fetch_and_index as MCP tools that any Claude session can call.
MCP multi-runtime JSON-RPC

// the meta-loop

Claude Builds
Itself a Better Tool

The most unusual part: CLAUDE.md instructs Claude to use context-mode to build context-mode. The tool optimizes the sessions that improve the tool.

CLAUDE.md — Project Instructions
These rules live in the repo. Claude reads them at session start and must follow them when working on context-mode itself — creating a self-referential feedback loop.
# Tool Selection

1. GATHER: batch_execute(commands, queries)
   — Primary tool for research. ONE call replaces many steps.

2. FOLLOW-UP: search(queries: ["q1", "q2"])
   — Use for all follow-up questions. ONE call, many queries.

3. PROCESSING: execute(language, code)
   — Use for API calls, log analysis, data processing.

# Rules
- DO NOT use Bash for commands producing >20 lines of output
- DO NOT use Read for analysis — use execute_file
- DO NOT use WebFetch — use fetch_and_index instead

Every significant technical decision in context-mode was designed, reviewed, or refined in dialogue with Claude — from initial architecture to final security hardening.

🏗️

Architecture Design

Designed the two-phase flow: execute-into-sandbox then query-from-sandbox, keeping raw output permanently out of the context window.

⚙️

Rust Executor

Wrote the sandboxed Rust executor with process group isolation, timeout enforcement, and clean teardown to prevent zombie processes.

🔒

Security Reviews

Identified and resolved security blockers B1–B4: path traversal, command injection, sandbox escape vectors, and resource exhaustion.

🧪

Test Coverage

Added coercion tests for string→array normalization in batch_execute, ensuring the API degrades gracefully for all input shapes.

🔄

The Meta-Loop

Drafted CLAUDE.md rules that instruct Claude to use context-mode while building context-mode — a self-improving feedback loop.

📖

API Ergonomics
